# AI Coder Instructions

## Mission

Build a production-ready chess study web application inspired by Lotus
Chess while remaining an original product.

## Principles

-   Modular architecture
-   Strong TypeScript typing
-   Reusable components
-   Accessibility
-   Unit tests where practical
-   No placeholder code
-   Document exported functions

## Milestones

1.  Initialize project
2.  Configure Firebase
3.  Authentication
4.  Chessboard
5.  Opening browser
6.  Opening trainer
7.  Endgame trainer
8.  Stockfish integration
9.  Spaced repetition
10. Dashboard
11. Search
12. Admin content tools
13. Polish and deployment

## Coding Standards

-   Feature-based folders
-   React hooks
-   Server Components where appropriate
-   Client components only when required
-   Keep business logic separate from UI
-   Avoid duplication
-   Maintain clean commit history

## Definition of Done

-   Feature implemented
-   Responsive
-   Tested
-   No console errors
-   Documented
-   Ready for production
