# Product Requirements Document

## Goal

Build a web application for mastering chess openings and endgames
through interactive practice, adaptive review, and progress tracking.

## MVP

-   Authentication
-   Opening library
-   Endgame library
-   Interactive trainer
-   Stockfish integration
-   Progress dashboard
-   Search
-   Responsive UI

## Functional Requirements

### Authentication

-   Google and Email login
-   Persist sessions

### Opening Trainer

-   Browse openings and variations
-   Interactive move-by-move practice
-   Immediate feedback
-   Explanations for mistakes
-   Mastery score per variation

### Endgame Trainer

-   Practice curated positions
-   Engine opponent
-   Hint system
-   Success verification

### Dashboard

-   XP
-   Streak
-   Rank
-   Accuracy
-   Time studied

### Acceptance Criteria

-   No illegal moves
-   Engine responds under 2 seconds on common hardware
-   Progress syncs via Firebase
-   Mobile and desktop responsive
