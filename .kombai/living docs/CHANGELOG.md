# Changelog

All notable changes to this project will be documented in this file.

## [0.2.0] - 2026-07-15
### Added
- **Cyber-Grandmaster Design System**: Obsidian background, glassmorphism, and neon accents.
- **Progress Dashboard**: 12-column bento grid layout with Elo history and Tactical Radar charts.
- **Real-Time Engine Interface**: Integrated Stockfish metrics (Nodes, Depth, NPS) into the global system status.
- **Unified Interactive Carousels**: 3-card rotating views implemented across both Library and Openings Practice sections.
- **Opening Snapshots**: Mini-chessboard views for repertoire variations.
- **Living Documentation**: ROADMAP, DECISIONS, TECH_DEBT, and CHANGELOG integration.

### Changed
- Rebranded application from "Lotus Chess" to "OE Chess Lab".
- Overhauled Opening Trainer UI for a higher-tech analytical feel.
- Migrated global styles to Tailwind v4.
- Standardized complexity/difficulty indicators across the platform.

## [0.1.0] - 2026-07-10
### Added
- Initial Chess Training application foundation.
- Firestore integration for user progress and openings.
- Basic Opening and Endgame trainers.
